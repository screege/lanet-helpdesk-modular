CONFIGURACIÓN DE GITHUB SECRETS - PASO A PASO
==============================================

PASO 1: IR A GITHUB
-------------------
1. Abre tu navegador
2. Ve a: https://github.com/screege/lanet-helpdesk-modular
3. Click en la pestaña "Settings" (arriba)
4. En el menú izquierdo, click en "Secrets and variables"
5. Click en "Actions"
6. Click en "New repository secret"

PASO 2: CREAR SECRET 1
---------------------
Name: VPS_HOST
Secret: (copia del archivo SECRET_1_VPS_HOST.txt)

PASO 3: CREAR SECRET 2
---------------------
Name: VPS_PORT
Secret: (copia del archivo SECRET_2_VPS_PORT.txt)

PASO 4: CREAR SECRET 3
---------------------
Name: VPS_USER
Secret: (copia del archivo SECRET_3_VPS_USER.txt)

PASO 5: CREAR SECRET 4
---------------------
Name: VPS_SSH_KEY
Secret: (copia del archivo SECRET_4_VPS_SSH_KEY.txt)

VERIFICACIÓN FINAL
-----------------
Deberías ver 4 secrets en la lista:
✅ VPS_HOST
✅ VPS_PORT
✅ VPS_USER
✅ VPS_SSH_KEY

¡LISTO! Una vez que tengas los 4 secrets configurados, 
podemos continuar con el setup del VPS.
